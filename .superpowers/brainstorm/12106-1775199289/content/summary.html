Audit and fix all broken internal navigation, missing image references, and any non-functioning buttons or links across the entire app.

Screen-by-screen audit of login, test list, test interface, review screen, completion screen, and teacher dashboard. Fix layout breaks, overlapping elements, clipped content.

Remove dead code from cleared assignments, clean up empty lesson structures, ensure Fuse.js only loads on demand. Smaller, faster page.

One-click export from Reports tab — all students or below-passing only. Includes name, assignment, score %, MC %, SR %, pass/fail, time spent, submission date.
